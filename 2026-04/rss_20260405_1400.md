# 【1】国产铁矿石短期稳中偏弱，区域分化格局持续

**来源**：我的钢铁铁矿石 | **时间**：2026-04-05 12:00
**链接**：[国产矿周报（3.30-4.3）](https://mp.weixin.qq.com/s/BbNeC9uehWkiDZRfDkUDqA)

## 核心观点

**价格走势分化**：本周华北、东北、华东区域精粉价格小幅回落（跌幅0.65%~1.03%），西南云南和西北哈密逆势上行（涨幅1.35%~2.39%）。核心驱动：连铁期货盘面震荡偏弱，贸易商挺价意愿松动，但西南/西北区域供给偏紧、钢厂复产补库支撑价格。

**供应端**：矿山产能利用率环比延续下行，华东华北为主要减量来源（部分大型矿企短期检修）；进口矿库存小幅上涨至1355万吨（+0.18%），国产矿库存基本平稳（78.15万吨，-0.15%）。

**钢材产量回升**：本周五大钢材品种供应851.51万吨，周环比增加11.93万吨（+1.4%），螺纹钢、线盘均由降转升，主因江苏、辽宁部分钢厂高炉复产。

## 投资方向

| 维度 | 观点 |
|------|------|
| 铁矿石期货 | 短期震荡偏弱，关注连铁波段机会 |
| 区域差异 | 西南/西北精粉供需偏紧，可关注相关资源股 |
| 钢铁板块 | 复产扩产趋势持续，关注江苏/辽宁区域龙头 |

**下周预测**：70%受访客户认为国产矿价格维持平稳，30%认为有下降空间；全部矿山/选厂客户认为价格暂稳。整体偏弱震荡。

# 【2】食饮行业：茅台提价提振白酒信心，啤酒旺季改善可期

**来源**：猛哥看商业（国泰海通食饮研究）| **时间**：2026-04-05 11:10
**链接**：[茅台提价提振信心，关注啤酒旺季改善](https://mp.weixin.qq.com/s/pskA5_7D7ltEZZ4xgwnM3A)

## 核心观点

### 白酒：茅台提价，行业信心回升

- 3月30日茅台宣布提价：普飞销售合同价提价 **8.6%**、自营零售价提价 **2.7%**
- 核心逻辑：随行就市+供需适配，夯实品牌定价权，倒逼黄牛出清
- 叠加 i茅台限制单日下单数，货源将更多流向真实消费者，有助提高开瓶率
- **短期关注**：酒企年报及一季报（业绩、回款指标），预计影响市场对周期底部判断
- **风险**：动销未确认企稳，库存和报表尚未出清，当前仍处淡季

### 啤酒：消费税规范化+旺季临近，头部份额提升逻辑增强

- 4月1日起，啤酒消费税以**出厂价与关联销售价孰高者**为计税标准（堵塞关联公司压价漏洞）
- 长期有利于规范经营的头部酒企（华润、青岛、百威等）份额提升
- 旺季临近，餐饮需求逐步回暖，低预期+低基数背景下关注景气弹性兑现

# 【3】原油专家深度交流：霍尔木兹封锁重塑全球能源格局，油价中枢系统性上移

**来源**：LittleLee的价值投机 | **时间**：2026-04-05 09:11
**链接**：[原油专家交流纪要](https://mp.weixin.qq.com/s/om4mKT9DNYQBcOMW3k5-tw)

## 核心观点

### 一、油价中枢系统性上移（最重要判断）

- 冲突持续时间远超预期，未来结束迹象不明朗
- **盘面价格（~110美元）≠真实能源成本**：真实现货价格（含升水+物流）可能已达 **120~130美元**
- 即便霍尔木兹海峡4月底恢复通行，全年能源损失量仍达约 **9亿桶**，补库需求延续至明年底
- **中长期油价中枢永久性抬升10~20美元**，回落至80美元以下将触发各国积极补库

### 二、原油缺口vs成品油缺口

| 维度 | 现状 |
|------|------|
| 原油实际缺口 | 约700万桶/日（沙特东西管道转运~500万桶/日+阿联酋管道~100万桶/日+伊朗~200万桶/日部分缓解） |
| 成品油缺口 | 约500万桶/日，几乎无法绕道替代，**柴油缺口最严重** |
| 影响地区 | 印尼、巴西、非洲、欧洲柴油短缺；东北亚/东南亚石脑油原料紧张 |

### 三、五月可能是最紧张时点

- 三月：恐慌；四月：靠SPR+库存+抢货"裱糊"；**五月：三重压力叠加（原料更贵+旺季需求+库存缓冲耗尽）**
- 届时不排除靠极端高价或行政手段压制终端需求

### 四、受益方向

- **上游资源型企业**：原油资源敞口大、炼厂业务占比低，受益油价中枢上移
- **煤化工企业**：不依赖高价进口原油，且下游替代品价格随油化工紧张而上移
- **能顺利出口的化工品**（如瓶片）：供需偏紧+出口窗口打开，成本向下游传导顺畅

### 五、注意风险

- 中国炼厂：原油采购成本持续攀升，但国内成品油价格传导受限+出口受限，利润承压
- 盘面与现货严重脱节：不能单看期货价格判断实际采购成本

## 投资方向

| 方向 | 逻辑 |
|------|------|
| A股上游石油企业（中石油/中海油等） | 资源端受益油价中枢上移 |
| 煤化工板块 | 原料成本稳定+替代品涨价逻辑 |
| 瓶片/出口化工品 | 出口利润扩张，传导链条顺畅 |
| 回避：中国炼厂 | 高成本+限价+限出口，利润压力大 |
| 关注：柴油相关 | 全球柴油最短缺，相关标的弹性大 |
